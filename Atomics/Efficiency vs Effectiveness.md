# Efficiency vs Effectiveness

| **EFFICIENCY: Doing Things RIGHT**                                                                               | **EFFECTIVENESS: Doing the RIGHT Things**                                       |
| ---------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| Getting maximum output from minimum input.                                                                       | Attaining organizational goals.                                                 |
| Focuses on PROCESS and resource use.                                                                             | Focuses on OUTCOMES and goal achievement.                                       |
| Measured as output/input ratio.                                                                                  | Measured as goal-attainment rate.                                               |
| **Example:** Maruti Suzuki's lean manufacturing has the highest output per employee in the Indian auto industry. | **Example:** ISRO achieving Mars orbit on the first attempt is a global record. |
| **Risk:** Efficiently doing the wrong things.                                                                    | **Risk:** Effectively but expensively i.e., ignoring resource costs.            |
