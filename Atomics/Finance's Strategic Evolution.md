# Finance's Strategic Evolution

From Scorekeeper to Strategist
- from 1980s to now CFO has gone from accounting and reporting to Capital allocation, investor narrative, digital transformation investment, ESG reporting

ESG finance and sustainable investing
- Global ESG-labeled assets reached US$30 trillion 
- SEBI mandates ESG reporting for the top 1000 indian companies

Fintech disruption of the finance function
- Stripe, Razarpay, Open Banking API are disrupting corporate treasury
- CFOs now manage hybrid banking relationship and real time cash visibility through API

Real time financial intelligence:
- SAP, or Oracle fusion let finance operate in real time, not at month end
- Predictive analytics replaces backward looking reporting
- AI forecasting model predictive revenue to within 2% accuracy

Finance has shifted from control function to an architecture function
- how value is created measured and communicated
