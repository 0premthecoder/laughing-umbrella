PPF downward sloping says we have to give up other product to generation of another one 
- Slope = [[opportunity cost]]


Some are attainable like point H which shows scarcity of resources

Point (A --> F) are truly efficient point which is attainable

Point (G) is inefficient 
- Not correct use of resources

In this diagram there is a bowed shape (we would call it non linear PFFs)
- Means that [[opportunity cost]] is not constant
- It is increasing here but could be decreasing
- It's reason could be Inputs are not perfectly homogenous
	- Some earlier inputs were better at B task 
	- But we are using them for A so thats the reason
	- Maybe The expansion can't be linear


Linear PPF would occur when [[opportunity cost]] is constant
