# What is a Sector

A sector is a broad segment of the economy containing multiple related industries 
Sector represents macroeconomic classification

| **Sector**               | **Included Industries**      |
| ------------------------ | ---------------------------- |
| **Manufacturing sector** | Automobiles, steel, textiles |
| **Services sector**      | Banking, hospitality, IT     |
| **Technology sector**    | Software, semiconductors, AI |
| **Healthcare sector**    | Hospitals, pharma, biotech   |
