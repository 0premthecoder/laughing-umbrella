# Managerial Competency Framework

Personal
- Managing self internally: self awareness, emotional regulation, resilience
- Managing self externally: time management, personal productivity, presence

Informational
- Communicating verbally: clarity, persuasion
- Communicating non-verbally: presence, listening
- Communicating analytically: data interpretation, judgement under uncertainity

Interpersonal
- leading individual: coaching, motivation
- leading group: team dynamics, facilitation
- leading the organization: culture, strategy and communication

Actional
- Scheduling: time allocation
- Administering: managing process
- Designing: Creating system
- Mobilizing: marshalling people and resources for action
